# 📌 [건강 상태 비교 프로그램]
 
## 📝 무엇을 하는 프로그램인가요?
국민건강보험공단의 건강검진 공공데이터를 활용하여 사용자의 건강 수치를 정상 기준과 비교하고, 같은 성별·연령대 집단 안에서의 위치를 그래프로 보여주는 GUI 프로그램
 
## ⚙️ 실행 방법
1. python 3.10 이상 설치
2. matplotlib 설치 (`pip install matplotlib`)
3. 공공데이터포털에서 건강검진정보 zip 파일 다운로드
3. 국민건강보험공단 건강검진정보 zip 파일을 코드 내 `zip_path` 경로에 맞게 위치시키기
   (코드 속 zip_path 변수를 본인 PC의 zip 파일 위치로 수정한 후 실행)
4. `python health_check_program.py` 실행
5. 프로그램이 실행되면 첫 화면에서 원하는 기능의 버튼 또는 종료를 클릭

### 1) 건강 판별 기능
1. 첫 화면에서 "건강 판별" 버튼 클릭
2. 이름, 키(cm), 몸무게(kg), 수축기혈압, 이완기혈압, 공복혈당 입력
3. "결과 확인" 버튼 클릭
4. 새 창에 BMI 수치와 BMI/혈압/혈당 판정 결과가 표시됨

### 2) 건강 분포 확인하기
1. 첫 화면에서 "건강 분포 확인하기" 버튼 클릭
2. 이름, 성별(남자=1 / 여자=2), 나이, 키(cm), 몸무게(kg), 수축기혈압, 이완기혈압, 공복혈당 입력
3. "결과 확인" 버튼 클릭
4. 새 창에 동일한 성별·연령대 그룹의 BMI/혈압/혈당 분포 그래프와, 각 항목의 평균·표준편차·내 위치(백분위수)가 표시됨

### 3) 종료
- 첫 화면에서 "종료" 버튼 클릭



## 🐍 사용된 Python 개념

- **함수 정의 및 호출**: `calculate_bmi`, `classify_bmi`, `classify_bp`, `classify_glucose` 등 기능별로 함수를 분리하여 작성하고 필요한 곳에서 호출
- **딕셔너리**: 건강검진 데이터를 `{'gender':..., 'age_group':..., 'height':..., ...}` 형태로 저장하여 `health_data` 리스트에 담음
- **리스트, 리스트 컴프리헨션**: `health_data`에서 같은 성별/연령대 데이터를 `filtered_data = [data for data in health_data if ...]`로 필터링, `bmi_list = [calculate_bmi(...) for data in filtered_data]`처럼 그래프용 데이터 추출
- **조건문(if/elif/else)**: BMI/혈압/혈당 판정 함수에서 구간별 분류, 입력값 유효성 검사(범위 체크)
- **반복문(for)**: csv 파일의 각 행(row)을 순회하며 데이터 정리, 통계 계산 시 합계 구하기(`sum(... for ... in ...)`)
- **예외 처리(try/except)**: 사용자가 숫자가 아닌 값을 입력했을 때 `ValueError`를 잡아서 오류 메시지 표시
- **파일 입출력**: `zipfile` 모듈로 zip 파일 열기, `csv.DictReader`로 CSV 파일을 한 줄씩 딕셔너리 형태로 읽기, `cp949` 인코딩 지정
- **f-string**: 결과 텍스트와 그래프 라벨을 동적으로 생성 (`f'{name}님의 값: {user_value}'` 등)
- **수학 계산**: 평균, 분산, 표준편차, 백분위수를 직접 수식으로 계산 (`sum`, `len`, 거듭제곱 연산자 `**`)

### tkinter에서 사용한 기능
- `tk.Tk()`: 메인 윈도우(앱의 본체 창) 생성
- `tk.Toplevel()`: 메인 창 위에 새 팝업 창(결과창, 그래프창) 생성
- `tk.Label()`: 텍스트(제목, 입력 안내, 결과, 통계) 표시
- `tk.Entry()`: 사용자 입력칸 생성, `.get()`으로 입력값 가져오기
- `tk.Button(command=...)`: 버튼 클릭 시 특정 함수 실행
- `tk.Frame()`: 화면을 영역별로 나누어 위젯을 그룹화 (그래프 3개를 가로로 배치할 때 사용)
- `.pack()`: 위젯을 화면에 배치하는 방법, `pady`/`padx`로 여백 조정, `side='left'`로 가로 정렬
- `winfo_children()` + `.destroy()`: 화면 전환 시 기존 위젯을 모두 제거 (`clear_window` 함수)
- `messagebox.showerror()`, `messagebox.showinfo()`: 오류/안내 팝업창 표시
- `root.mainloop()`: 프로그램이 사용자 입력을 계속 기다리도록 이벤트 루프 실행

### matplotlib에서 사용한 기능
- `plt.subplots(figsize=...)`: 그래프(figure)와 좌표축(axes) 생성, 크기 지정
- `ax.hist(data, bins=20, color=..., alpha=...)`: 데이터 분포를 히스토그램으로 표현 (혈압 그래프는 두 개의 히스토그램을 겹쳐서 표시)
- `ax.axvline(value, color=..., linestyle=...)`: 사용자의 값을 그래프 위에 수직선으로 표시
- `ax.set_title()`, `ax.set_xlabel()`, `ax.set_ylabel()`: 그래프 제목과 축 이름 설정
- `ax.set_xlim()`: x축 표시 범위 제한 (혈당 그래프)
- `ax.legend()`: 범례 표시
- `fig.tight_layout()`: 그래프 요소 간 간격을 자동으로 보기 좋게 조정
- `FigureCanvasTkAgg(fig, master=...)`: matplotlib 그래프를 tkinter 위젯으로 변환
- `canvas.draw()`: 그래프 렌더링
- `canvas.get_tk_widget().pack()`: 변환된 그래프를 tkinter 화면에 배치
