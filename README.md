# [프로젝트 제안서] - 통계학과 김민주(25)

## 1. 📌 프로젝트 이름

건강검진 통계 비교 프로그램

## 2. 📝 한 문장 설명

국민건강보험공단의 건강검진 공공데이터를 활용하여 사용자의 건강 수치를 정상 기준과 비교하고, 같은 성별·연령대 집단 안에서의 위치를 그래프로 보여주는 GUI 프로그램이다.

## 3. ❓ 풀려는 문제 또는 만들려는 이유

병원 복도에 비치된 혈압측정기로 혼자 혈압을 측정했을 때, 수치가 정상 범위에 해당하는지 판단하기 어려웠던 경험이 있다.
이에 혈압, BMI, 혈당 등 개인 건강 수치를 입력하면 정상 여부를 자동으로 판별하고, 나아가 비슷한 조건의 사람들과 비교하여 자신의 건강 상태의 대략적인 위치를 파악할 수 있는 프로그램을 만들고자 한다.

> ⚠️ 주의: 이 프로그램은 의학적 진단이 아니라 건강 수치를 참고용으로 확인하기 위한 프로그램이다.

## 4. ⚙️ 핵심 기능 3가지

1. **건강 수치 자가 진단 기능**
   - 사용자가 키, 몸무게, 수축기 혈압, 이완기 혈압, 공복혈당을 입력하면, 키와 몸무게 값을 통해 BMI를 계산한다.
   - BMI, 혈압, 공복혈당을 일반적인 건강 기준 범위와 비교하여 정상, 주의, 위험 여부를 안내한다.

2. **동일 집단 추출 및 평균 계산 기능**
   - 사용자가 선택한 성별과 연령대를 기준으로 국민건강보험공단 건강검진 공공데이터에서 비슷한 조건의 집단을 추출한다.
   - 추출된 집단의 평균 BMI, 평균 혈압, 평균 공복혈당을 계산한다.

3. **건강 수치 분포 시각화 및 내 위치 표시 기능**
   - 같은 성별·연령대 집단의 BMI, 혈압, 공복혈당 분포를 각각 그래프로 나타낸다.
   - 그래프 위에 사용자의 수치를 표시하여 자신의 건강 수치가 집단 내에서 어느 정도 위치에 있는지 시각적으로 확인할 수 있도록 한다.

## 👤 5. 예상 사용자

건강검진 대상이 되는 성인

## 🖥️ 6. 결과물 형태

GUI 프로그램

Python의 tkinter를 사용하여 첫 화면에 `건강 수치 자가 진단`과 `나의 건강 위치` 두 개의 메뉴 버튼을 배치할 예정이다.  
`건강 수치 자가 진단`에서는 사용자가 입력한 건강 수치를 기준 범위와 비교한다.  
`나의 건강 위치`에서는 성별과 연령대를 선택한 뒤 공공데이터에서 같은 조건의 집단을 추출하고, BMI, 혈압, 공복혈당의 분포 그래프 위에 사용자의 위치를 표시한다.

## 🐍 7. 사용할 Python 개념

| Python 개념 | 활용 방식 |
|---|---|
| 변수 | 사용자의 키, 몸무게, 혈압, 혈당 등 입력값을 저장한다. |
| 조건문 | BMI, 혈압, 공복혈당이 기준 범위에 속하는지 판별한다. |
| 리스트 | 공공데이터에서 추출한 건강 수치들을 저장한다. |
| 딕셔너리 | 한 사람의 건강검진 정보를 항목별로 저장하거나 관리한다. |
| 반복문 | CSV 데이터의 여러 행을 반복해서 확인하고 조건에 맞는 데이터를 찾는다. |
| 함수 | BMI 계산 및 정상 기준 판별, 평균 계산, 그래프 출력 기능을 나누어 구현한다. |
| CSV 모듈 | CSV 파일을 불러온 후 행 단위로 읽고, 각 행을 딕셔너리 형태로 처리한다. |
| tkinter | 입력창, 버튼, 결과창이 있는 GUI 화면을 구성한다. |
| matplotlib | 그래프 및 분포를 표현한다. |
| 예외처리 | 사용자가 비정상적인 값을 입력했을 때 발생하는 오류를 방지하기 위해 사용한다.|
